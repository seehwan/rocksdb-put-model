2) 핵심 구성요소

2.1 Device Envelope Modeling

구현 파일: model/envelope.py

기능: 4D 그리드 스윕을 통한 실제 장치 특성 반영

핵심 수식:
Beff = Envelope(ρr, qd, numjobs, bs; Θ_device)
그리드 스윕 범위:
  • ρr ∈ {0, 25, 50, 75, 100}% (읽기 비율)
  • iodepth ∈ {1, 4, 16, 64} (큐 깊이)
  • numjobs ∈ {1, 2, 4} (병렬 작업 수)
  • bs ∈ {4, 64, 1024} KiB (블록 크기)
주요 기능:
  • 4D 선형보간 (scipy.interpolate.RegularGridInterpolator)
  • 물리적 제약 클램프 (Beff ≤ min(Br, Bw))
  • 외삽 경고 시스템
  • 물리적 제약 검증
테스트 통과

2.2 Closed Ledger Accounting

구현 파일: model/closed_ledger.py

기능: 물리적 검증을 통한 회계 폐곡선 보장

표준화된 회계 정의:
WA_stat = (WAL + Flush + Σ CompWrite) / UserWrite WA_device = DeviceWrite / UserWrite RA_comp = Σ CompRead / UserWrite RA_runtime = DeviceRead / UserWrite
폐곡선 검증:
|WA_stat - WA_device| ≤ 10%
주요 기능:
  • RocksDB LOG 파싱
  • Statistics JSON 파싱
  • iostat 데이터 통합
  • Per-level I/O 분석
  • 폐곡선 검증
테스트 통과

2.3 Dynamic Simulation Framework

구현 파일: model/v4_simulator.py

기능: 시간가변 시스템 동작 모델링

핵심 수식:
# Per-level 용량 계산 C_ℓ(t) = μ_ℓ · k_ℓ · η_ℓ(t) · Beff(t) # Backlog 동역학 Q_ℓ(t+Δ) = max{0, Q_ℓ(t) + I_ℓ(t) - C_ℓ(t)·Δ} # Stall 확률 p_stall = σ(a · (N_L0(t) - τ_slow))
주요 기능:
  • 동적 시뮬레이션 엔진
  • Per-level 용량 제약
  • Backlog 동역학 모델링
  • Stall 확률 계산
  • Read ratio 추정
  • L0 파일 동역학

7) 사용 방법

7.1 기본 시뮬레이션 실행

# 전체 테스트 실행 python3 test_v4_model.py # 결과: 4/4 테스트 통과, 성능 메트릭 분석

7.2 커스텀 시뮬레이션

from model.envelope import create_sample_envelope_model from model.v4_simulator import V4Simulator import yaml # 엔벌롭 모델 생성 envelope = create_sample_envelope_model() # 설정 로드 with open('config/v4_simulator_config.yaml', 'r') as f: config = yaml.safe_load(f) # 시뮬레이터 생성 및 실행 simulator = V4Simulator(envelope, config) results = simulator.simulate(steps=1000, dt=1.0) # 결과 분석 analysis = simulator.analyze_results(results)

7.3 Device Envelope 측정

# fio 그리드 스윕 실행 ./tools/device_envelope/run_envelope.sh # 결과 파싱 python3 tools/device_envelope/parse_envelope.py device_envelope_results/

7.4 Closed Ledger 분석

from model.closed_ledger import ClosedLedger # Ledger 인스턴스 생성 ledger = ClosedLedger() # RocksDB 데이터 처리 ledger_data = ledger.process_rocksdb_data( log_path='rocksdb.log', stats_path='statistics.json' ) # 폐곡선 검증 verification = ledger.verify_ledger_closure(ledger_data['wa_ra_data'])
