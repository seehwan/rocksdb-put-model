0) 실험 개요

실험 환경

  • 하드웨어: Intel i7-12700K, 32GB RAM, Samsung 980 PRO 2TB NVMe SSD
  • 소프트웨어: Ubuntu 22.04 LTS, RocksDB 8.0.0, Python 3.10
  • 실험 기간: 2025년 9월 5일 - 9월 7일 (3일간)
  • 총 실험 수: 180개 fio 실행 + 50개 RocksDB 벤치마크

실험 목표

  • Device Envelope 정확성: 4D 그리드 모델의 실제 장치 특성 반영도
  • Closed Ledger 검증: 회계 폐곡선의 물리적 정확성
  • Dynamic Simulation 성능: 시간가변 시스템 동작 예측 정확도
  • v3 대비 개선: 기존 모델 대비 성능 향상 정도

성공 기준

  • Ledger Closure: |WA_stat - WA_device| ≤ 10%
  • Holdout MAPE: ≤ 15%
  • Envelope Accuracy: 보간 오차 ≤ 5%
  • Simulation Speed: ≥ 1000 steps/sec
